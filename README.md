# kekforce-card-dispenser
Description: Interactive display of the KEKFORCE card, an official fake rare card on Counterparty (XCP) on Bitcoin. This repo showcases the card’s visuals and audio while providing real-time information on the last sale and current open dispenser status. Perfect for sharing on platforms like Twitter.

KEKFORCE Card Dispenser

Welcome to the interactive display of the KEKFORCE card!

Features

	•	Interactive Hover: Hover over the card to flip and view details.
	•	Audio Integration: Experience the unique sound of the KEKFORCE card.
	•	Real-time Information: See the last sale price and the current status of the dispenser at a glance.

How to Use

	1.	Clone the repository or view it live here.
	2.	Hover over the card to interact.
	3.	Enjoy the immersive experience of KEKFORCE!

Technologies Used

	•	HTML, CSS, and JavaScript for frontend interactivity.
	•	Counterparty (XCP) on Bitcoin for the underlying blockchain details.
